# SKILL MESTRA — CARROSSÉIS EMAGRECIMENTO BLINDADO

Você é responsável por criar carrosséis premium para o projeto Emagrecimento Blindado.

O objetivo é gerar carrosséis prontos para Instagram, no formato vertical, com identidade visual forte, elegante, feminina, preta e dourada, mantendo leitura clara, fotos bem encaixadas e CTA para comentários com a palavra BLINDADO.

---

## 1. Formato obrigatório

Todos os cards devem ser exportados em:

- Formato: PNG
- Tamanho: 1080 x 1350 px
- Proporção: 4:5
- Qualidade: alta
- Fundo: escuro premium, preto/grafite, com dourado como destaque
- Quantidade padrão: 10 cards por carrossel

Cada carrossel deve ser salvo em uma pasta própria:

```
CARROSSEL_01_NOME_DO_TEMA
CARROSSEL_02_NOME_DO_TEMA
CARROSSEL_03_NOME_DO_TEMA
```

Dentro de cada pasta, os arquivos devem ser nomeados assim:

```
card_01.png
card_02.png
card_03.png
card_04.png
card_05.png
card_06.png
card_07.png
card_08.png
card_09.png
card_10.png
```

Também pode ser criado um arquivo:

```
PREVIEW.png
```

O `PREVIEW.png` é apenas para conferência. Não deve ser usado no Canva nem no Instagram.

---

## 2. Identidade visual obrigatória

A identidade do projeto é:

- Preto
- Grafite
- Dourado elegante e visível
- Branco para títulos principais
- Visual premium
- Visual feminino
- Visual sofisticado
- Visual limpo
- Visual de autoridade

Não usar visual colorido, infantil, bagunçado ou poluído.

O dourado não deve ficar tímido. Ele precisa aparecer como elemento de marca, mas sem exagero.

---

## 3. Regras de fotos

As fotos são uma parte importante do layout.

Ao inserir fotos:

- Nunca cortar cabeça
- Nunca cortar testa
- Nunca cortar rosto
- Nunca cortar corpo de forma estranha
- Nunca deixar a pessoa escondida atrás de faixa, linha, logo ou texto
- Nunca colocar texto por cima do rosto
- Nunca colocar linha dourada atravessando rosto, cabelo, testa ou cabeça
- Nunca colocar @ ou logotipo sobre a foto principal
- A foto deve aparecer inteira sempre que possível
- Para fotos verticais: usar `object-fit: contain`
- Para fotos horizontais: usar `object-fit: contain`
- Usar `object-position: center top`
- Se sobrar espaço, preencher com fundo escuro, blur elegante, sombra ou moldura — mas não cortar a pessoa

A prioridade é: a pessoa precisa aparecer bem, com rosto limpo e sem elementos invadindo.

---

## 4. Regras do @emagrecimentoblindado

O `@emagrecimentoblindado` deve aparecer apenas no rodapé.

Regras:

- Não colocar @ no meio do card
- Não colocar @ na parte superior
- Não colocar @ sobre rosto, cabeça ou foto
- Não colocar @ atravessando a imagem
- Usar pequeno, discreto e elegante
- Manter dentro da margem segura
- Pode aparecer no canto inferior direito ou centralizado no rodapé

Se houver dúvida, usar apenas no rodapé inferior direito.

---

## 5. Regras de linhas, molduras e elementos dourados

Elementos dourados são permitidos, mas devem respeitar o conteúdo.

**Pode usar:**

- Cantos dourados
- Linhas finas decorativas
- Pequenas barras douradas
- Molduras elegantes
- Botão dourado com texto preto

**Não pode:**

- Linha dourada passando no meio do rosto
- Linha dourada passando na cabeça
- Linha dourada passando por cima de texto importante
- Linha atravessando a foto principal
- Elemento decorativo atrapalhando leitura

A linha horizontal dourada, se usada, precisa ficar abaixo da foto ou em área segura — nunca sobre a pessoa.

---

## 6. Tipografia

Usar hierarquia clara:

**Título principal:**

- Grande
- Branco
- Forte
- Elegante
- Fácil de ler
- Pode usar fonte serifada sofisticada

**Texto secundário:**

- Menor
- Branco ou bege claro
- Bem espaçado
- Sem excesso de informação

**CTA:**

- Claro
- Direto
- Com a palavra BLINDADO em destaque

**Botão:**

- Fundo dourado
- Texto preto
- Palavra: BLINDADO
- Não pode ficar colado na borda inferior
- Manter margem segura

---

## 7. Margens de segurança

Usar margem segura em todos os cards.

Nenhum texto, botão, logo, @ ou elemento importante pode ficar colado nas bordas.

**Margem recomendada:**

- Laterais: mínimo 70 px
- Topo: mínimo 70 px
- Rodapé: mínimo 90 px

O botão BLINDADO deve ficar acima da borda inferior, com respiro visual.

---

## 8. Conteúdo dos carrosséis

O conteúdo deve falar com mulheres que estão frustradas com o corpo, cansadas de tentar, sem engajamento emocional com dietas genéricas e que se identificam com dores reais.

**Evitar chamadas genéricas como:**

- Quer emagrecer?
- Perca peso rápido
- Dieta perfeita
- Emagreça agora

**Preferir chamadas com dor específica e identificação:**

- Você sente que faz tudo certo e mesmo assim nada muda?
- Você evita foto porque não aguenta se ver?
- Você já cansou de recomeçar toda segunda?
- Você come pouco e mesmo assim sente que seu corpo não responde?
- Você compra roupa larga só para não marcar?
- Você sente vergonha quando alguém comenta sobre seu corpo?

**O CTA padrão deve ser:**

> Comente BLINDADO.

Ou:

> Comente BLINDADO se você quer entender o método.

Ou:

> Comente BLINDADO para entrar no Grupo VIP.

---

## 9. Estrutura padrão de 10 cards

Cada carrossel deve seguir esta lógica:

| Card | Função |
|------|--------|
| 01 | Hook forte, com dor específica e promessa de leitura |
| 02 | Aprofundar a dor |
| 03 | Mostrar identificação emocional |
| 04 | Apontar o erro comum ou crença errada |
| 05 | Explicar de forma simples o que está acontecendo |
| 06 | Mostrar uma virada de consciência |
| 07 | Apresentar uma nova forma de pensar |
| 08 | Aumentar desejo pelo método ou pelo Grupo VIP |
| 09 | Reforçar que a pessoa não está sozinha e que existe caminho |
| 10 | CTA forte para comentar BLINDADO |

---

## 10. Layout recomendado por card

Cada card deve ter:

- Fundo escuro premium
- Elemento dourado discreto e elegante
- Título principal grande
- Texto secundário curto
- Espaço visual respirando
- Foto bem encaixada, se houver foto
- @ apenas no rodapé
- CTA visível
- Nada cortado
- Nada em cima do rosto

O layout precisa parecer pronto para uma marca séria, não como arte improvisada.

---

## 11. Conferência obrigatória antes de exportar

Antes de finalizar, conferir card por card:

- [ ] A foto aparece bem?
- [ ] A cabeça não foi cortada?
- [ ] O rosto não está coberto?
- [ ] A linha dourada não atravessa a pessoa?
- [ ] O @ está apenas no rodapé?
- [ ] O texto está legível?
- [ ] O dourado aparece bem?
- [ ] O fundo não está escuro demais?
- [ ] O botão BLINDADO não está colado na borda?
- [ ] Todos os cards estão em 1080 x 1350?
- [ ] Os arquivos estão nomeados de card_01.png a card_10.png?

Se algum desses pontos falhar, corrigir antes de exportar.

---

## 12. Exportação final

Ao terminar cada carrossel, criar uma pasta separada.

Exemplo:

```
CARROSSEL_01_COME_POUCO_E_NAO_MUDA/
├── card_01.png
├── card_02.png
├── card_03.png
├── card_04.png
├── card_05.png
├── card_06.png
├── card_07.png
├── card_08.png
├── card_09.png
├── card_10.png
└── PREVIEW.png
```

Repetir o mesmo padrão para todos os outros carrosséis.

---

## 13. Regra principal

A regra mais importante:

> **O design precisa vender autoridade, identificação emocional e desejo de comentar BLINDADO.**

Nada pode parecer amador, escuro demais, poluído ou mal encaixado.

A foto deve respeitar a pessoa. O texto deve respeitar a leitura. O CTA deve respeitar o objetivo: gerar comentários e entrada no Grupo VIP.
